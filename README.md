# NRG Vending Website

Welcome to the official repository for **NRG Vending**! This is a modern, one-page React website showcasing our vending machines and services.

## About NRG Vending
NRG Vending provides hassle-free vending solutions for gyms, workplaces, and laundromats in the Albury-Wodonga area. Our machines offer snacks, drinks, and contactless payments. We handle installation, stocking, and maintenance so you don’t have to worry.

## Features
- Modern, sleek design
- Responsive for mobile and desktop
- Contactless payment demo
- Easy to update and maintain
